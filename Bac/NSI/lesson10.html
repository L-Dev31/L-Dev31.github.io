1. Introduction au langage SQL

Qu'est-ce que SQL ?

SQL (Structured Query Language) est un langage permettant d'interagir avec des bases de données relationnelles. Il permet de créer, lire, modifier et supprimer des données dans des tables.

Exemple de table : Élèves
id_eleve nom prenom classe age
1 Martin Alice TermNSI 17

2. Opérations principales en SQL

Les grandes familles d'opérations
  • Lecture : extraire des données d'une table
  • Ajout : ajouter de nouvelles données
  • Modification : mettre à jour des données existantes
  • Suppression : retirer des données
Pourquoi utiliser SQL ?

SQL est utilisé dans la majorité des applications pour gérer les données de façon fiable, structurée et efficace.

3. Jointures et relations

Les jointures

Les jointures permettent de relier plusieurs tables entre elles pour croiser les informations (par exemple, relier les élèves et leurs notes).

Exemple de relation entre deux tables

On peut relier la table Élèves à la table Notes grâce à l'identifiant de l'élève.
