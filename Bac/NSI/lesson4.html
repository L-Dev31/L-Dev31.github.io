1. Tri par insertion

Principe

Le tri par insertion construit la liste triée élément par élément, en insérant chaque nouvel élément à sa place dans la partie déjà triée.

def tri_insertion(liste): """Tri par insertion - O(n²)""" for i in range(1, len(liste)): cle = liste[i] j = i - 1 # Décaler les éléments plus grands vers la droite while j >= 0 and liste[j] > cle: liste[j + 1] = liste[j] j -= 1 # Insérer la clé à sa position liste[j + 1] = cle return liste # Exemple d'utilisation nombres = [64, 34, 25, 12, 22, 11, 90] print("Liste non triée:", nombres) tri_insertion(nombres) print("Liste triée:", nombres)

2. Tri par sélection

Algorithme
  1. Trouver le minimum dans la liste
  2. L'échanger avec le premier élément
  3. Répéter avec le reste de la liste
def tri_selection(liste): """Tri par sélection - O(n²)""" n = len(liste) for i in range(n): # Trouver l'index du minimum min_idx = i for j in range(i + 1, n): if liste[j] < liste[min_idx]: min_idx = j # Échanger les éléments liste[i], liste[min_idx] = liste[min_idx], liste[i] return liste # Exemple avec animation des étapes def tri_selection_verbose(liste): print(f"Liste initiale: {liste}") n = len(liste) for i in range(n): min_idx = i for j in range(i + 1, n): if liste[j] < liste[min_idx]: min_idx = j if min_idx != i: liste[i], liste[min_idx] = liste[min_idx], liste[i] print(f"Étape {i+1}: {liste}") return liste

3. Tri fusion (Merge Sort)

Principe "Diviser pour régner"

Le tri fusion divise récursivement la liste en deux, trie chaque moitié, puis fusionne les résultats triés.

def tri_fusion(liste): """Tri fusion - O(n log n)""" if len(liste) <= 1: return liste # Diviser milieu = len(liste) // 2 gauche = tri_fusion(liste[:milieu]) droite = tri_fusion(liste[milieu:]) # Conquérir (fusionner) return fusionner(gauche, droite) def fusionner(gauche, droite): """Fusionne deux listes triées""" resultat = [] i = j = 0 # Comparer et fusionner while i < len(gauche) and j < len(droite): if gauche[i] <= droite[j]: resultat.append(gauche[i]) i += 1 else: resultat.append(droite[j]) j += 1 # Ajouter les éléments restants resultat.extend(gauche[i:]) resultat.extend(droite[j:]) return resultat # Exemple d'utilisation nombres = [38, 27, 43, 3, 9, 82, 10] print("Tri fusion:", tri_fusion(nombres))
Comparaison des complexités
Algorithme Complexité Stabilité
Tri insertion O(n²) ✓ Stable
Tri sélection O(n²) ✗ Instable
Tri fusion O(n log n) ✓ Stable
