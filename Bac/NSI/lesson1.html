1. Les listes (list)

Définition

Une liste est une structure de données ordonnée et modifiable qui peut contenir des éléments de types différents.

# Création et manipulation de listes ma_liste = [1, 2, "hello", 3.14, True] nombres = [10, 20, 30, 40, 50] # Accès aux éléments (indexation) print(ma_liste[0]) # 1 (premier élément) print(ma_liste[-1]) # True (dernier élément) # Modification ma_liste[1] = "modifié" print(ma_liste) # [1, 'modifié', 'hello', 3.14, True]
Opérations essentielles
  • append(x) : ajoute x à la fin O(1)
  • insert(i, x) : insère x à l'index i O(n)
  • remove(x) : supprime la première occurrence de x O(n)
  • pop(i) : supprime et retourne l'élément à l'index i O(n)

2. Les tuples (tuple)

Définition

Un tuple est une structure de données ordonnée et immutable (non modifiable).

# Création de tuples coordonnees = (3, 4) point = (10.5, 20.3, "origine") singleton = (42,) # Attention à la virgule ! # Accès (comme les listes) x, y = coordonnees # Déballage (unpacking) print(f"x = {x}, y = {y}") # Tuples immuables # coordonnees[0] = 5 # ERREUR !
Cas d'usage typiques
  • Coordonnées géographiques : (latitude, longitude)
  • Retour multiple de fonction : return (min_val, max_val)
  • Clés de dictionnaire : dico[(x, y)] = valeur

3. Les dictionnaires (dict)

Définition

Un dictionnaire est une structure de données qui associe des clés à des valeurs. Ordonné depuis Python 3.7.

# Création et manipulation etudiant = { "nom": "Dupont", "age": 17, "notes": [15, 18, 12], "mention": True } # Accès et modification print(etudiant["nom"]) # "Dupont" etudiant["age"] = 18 # Modification etudiant["classe"] = "Terminale" # Ajout # Méthodes importantes print(etudiant.keys()) # dict_keys(['nom', 'age', ...]) print(etudiant.values()) # dict_values(['Dupont', 18, ...]) print(etudiant.items()) # dict_items([('nom', 'Dupont'), ...])
Complexités importantes
  • Accès par clé : O(1)
  • Insertion/Suppression : O(1)
  • Recherche de valeur : O(n)
