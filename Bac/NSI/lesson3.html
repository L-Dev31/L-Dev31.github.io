1. Introduction aux arbres binaires

Définition

Un arbre binaire est une structure de données hiérarchique où chaque nœud a au maximum deux enfants : un enfant gauche et un enfant droit.

# Implémentation d'un nœud d'arbre binaire class Noeud: def __init__(self, valeur): self.valeur = valeur self.gauche = None self.droit = None # Création d'un arbre simple racine = Noeud(10) racine.gauche = Noeud(5) racine.droit = Noeud(15) racine.gauche.gauche = Noeud(2) racine.gauche.droit = Noeud(7)

2. Parcours d'arbres binaires

Parcours en profondeur
  • Préfixe (préordre) : Racine → Gauche → Droit
  • Infixe (en ordre) : Gauche → Racine → Droit
  • Postfixe (postordre) : Gauche → Droit → Racine
# Parcours préfixe (récursif) def parcours_prefixe(noeud): if noeud is not None: print(noeud.valeur) # Traiter la racine parcours_prefixe(noeud.gauche) # Parcourir à gauche parcours_prefixe(noeud.droit) # Parcourir à droite # Parcours infixe (récursif) def parcours_infixe(noeud): if noeud is not None: parcours_infixe(noeud.gauche) # Parcourir à gauche print(noeud.valeur) # Traiter la racine parcours_infixe(noeud.droit) # Parcourir à droite # Parcours en largeur (avec une file) from collections import deque def parcours_largeur(racine): if racine is None: return file = deque([racine]) while file: noeud = file.popleft() print(noeud.valeur) if noeud.gauche: file.append(noeud.gauche) if noeud.droit: file.append(noeud.droit)

3. Arbres binaires de recherche (ABR)

Propriété de l'ABR

Dans un ABR, pour chaque nœud : tous les éléments du sous-arbre gauche sont inférieurs à la valeur du nœud, et tous les éléments du sous-arbre droit sont supérieurs.

# Recherche dans un ABR def rechercher(noeud, valeur): if noeud is None or noeud.valeur == valeur: return noeud if valeur < noeud.valeur: return rechercher(noeud.gauche, valeur) else: return rechercher(noeud.droit, valeur) # Insertion dans un ABR def inserer(noeud, valeur): if noeud is None: return Noeud(valeur) if valeur < noeud.valeur: noeud.gauche = inserer(noeud.gauche, valeur) elif valeur > noeud.valeur: noeud.droit = inserer(noeud.droit, valeur) return noeud # Utilisation abr = None abr = inserer(abr, 10) abr = inserer(abr, 5) abr = inserer(abr, 15) abr = inserer(abr, 3)
Complexité des opérations
  • Recherche : O(log n) en moyenne, O(n) au pire cas
  • Insertion : O(log n) en moyenne, O(n) au pire cas
  • Suppression : O(log n) en moyenne, O(n) au pire cas
  • Parcours : O(n) toujours
