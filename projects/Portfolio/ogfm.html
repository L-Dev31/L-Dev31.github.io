The Brief

Project: Complete brand identity system for OG FM's 30th anniversary campaign. Scope: Logo system (mark + logotype), comprehensive brand charter, 3 print executions, 1 social media asset. Challenge: Create distinctive visual identity for underground DIY radio with near-zero brand visibility, targeting a hyper-niche nostalgia-driven audience (aged 30–60+). The brief: “Pour les jeunes cons qui sont devenus vieux” (For the young fools who grew old). A station for those who refuse to let the golden years disappear. The identity had to feel urgent, physical, unapologetic. Something that couldn't be mistaken for any other webradio.

Punk isn’t the absence of a system. It’s a system that refuses everyone else’s.

The Problem

Identity Gap

OG FM's original logo was generic and forgettable. Indistinguishable from any other online radio. For a station built on cultural defiance and nostalgia, the visual identity conveyed none of its actual personality. The radio existed in cultural invisibility. The identity system needed to be: (1) immediately recognizable, (2) ownable, impossible to confuse with competitors, (3) aligned with DIY/underground radio culture, and (4) defensible across both digital and print applications (social media, posters, anniversary campaign materials).

The Process

1. Strategic Foundation: Aesthetic Direction

Research identified UK DIY and fanzine culture as the core visual language. But with a constraint: most contemporary "punk" branding fakes this aesthetic in design software, creating visually authentic but emotionally hollow work. The strategic decision was to earn authenticity through analog-first methodology, making the execution inseparable from the concept.

2. Logo Development: Analog-First Execution

The logo was hand-drawn using ballpoint pen on aged paper, then cut out by hand. No digital grid, no undo button, no vector smoothing. This constraint-based process generated authentic imperfections that couldn't be replicated in software. The resulting mark was then scanned and vectorized in Illustrator, preserving the irregularities as a core design feature. Result: a mark that felt earned rather than stylized.

3. Type System & Brand Charter

The logotype was built using a 7-typeface system. Each selected to convey different eras and cultural references (punk, 80s rock, DIY fanzines, contemporary underground). Rather than a single "correct" font, the system established rules for how these fonts coexist: size hierarchy, spacing ratios, stretch/compress parameters, and offset values. This created a defensible, reproducible system that could be applied across 30+ social posts, posters, and print materials while maintaining visual coherence. The brand charter documented: logo variations (horizontal, stacked, icon-only), color palette (2 primary, 3 secondary), typography rules, spacing grid, and usage guidelines.

4. Campaign Execution & Deliverables

Delivered for 30th Anniversary Campaign: (1) 1 social media asset (Instagram post optimized for feed + stories dimensions), (2) 3 print executions (A2 poster, A4 flyer, merchandise label), (3) comprehensive brand charter (12-page PDF). The identity system was designed to scale: consistent application of logo, typography rules, color palette, and grid structure across all touchpoints. Every asset maintained the visual DNA: urgent, unapologetic, immediately recognizable. While varying execution to prevent monotony. The system demonstrated how constraint-based design rules enable flexibility rather than limiting it.

A strong identity system isn’t built by breaking rules randomly. It’s built by establishing new rules and defending them.

Key Learnings & Skills Demonstrated

Deliverables: Complete brand identity system including logo (symbol + logotype), comprehensive brand charter, 3 print executions, 1 social media asset. Skills applied: Strategic brand positioning, constraint-based design methodology, typography system design, design documentation, cross-platform asset application, vector design (Adobe Illustrator). Primary insight: Consistency and rule-based systems don't constrain creativity. They amplify it. The rigid framework enables flexible execution because stakeholders understand the underlying logic. A strong identity system is defensible, scalable, and reproducible.
